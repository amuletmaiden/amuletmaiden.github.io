Four-Color Text Editor
Click a color to set what newly typed text becomes. Select text to recolor it, apply repeating patterns, use bold or italics, switch blocks to H1 or paragraph, and paste HTML into the lower box to load it back into the editor.
Custom Pattern mode Typing color: red · Pattern mode: character
Ctrl+Z and Ctrl+Y use the editor’s own history stack. Character mode colors one visible character at a time. Word + punctuation mode colors each word as one hit and each punctuation mark as its own hit. Spaces and other whitespace never advance the pattern.
